FreeMASTER Serial Communication Driver for HC12/HCS12/HCS12X/S12Z Microcontrollers
================================================================================

This folder contains the HC12/HCS12/HCS12X/S12Z platform-dependent part of 
FreeMASTER serial communication driver. Please see the User Manual for 
more information about how to configure and use the driver.

Driver example and test applications available in <install_path>/examples folder.
